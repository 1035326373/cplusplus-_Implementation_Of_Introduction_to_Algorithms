设被比较序列\+S的长度为len1，比较序列\+T的长度为len2，则match算法执行一一比较来计算匹配位置：


\begin{DoxyItemize}
\item 对\+S，遍历其从 0$\sim$len1-\/len2的位置，在每一个位置上比较
\item 比较时，假设\+S的当前位置为i,则比较 S\mbox{[}i,i+len2-\/1\mbox{]}与\+T。当二者每一个字符都相等时，则匹配
\item 将找到的每一个匹配点存入std\+::vector中然后返回
\end{DoxyItemize}
